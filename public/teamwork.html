Team Collaboration - SLI Project

This webpage is part of a collaborative project on **Project Management**, developed to serve as a dummy website for the **Servant-Leader Institute (SLI)**. The success of this project would not have been possible without the hard work and collaboration of each team member. Effective teamwork was the cornerstone of this project, showcasing the importance of communication, shared responsibility, and leadership in achieving a common goal.

Each team member contributed their unique skills, from web development, marketing, to project management and content creation, demonstrating the importance of cross-functional collaboration in any project. This site stands as a testament to how critical team collaboration is in making any project a success.

Tools and Applications Used in the Project

Throughout the development of this project, we utilized various tools and applications to streamline collaboration, project management, and design. Below is a list of the tools that made this project possible:

  • Notion - Notion is best used for personal organization, team collaboration, project management, note-taking, knowledge management, and task tracking.
  • Google Docs - For uploading and downloading documentation and real-time content collaboration.
  • Figma - For UI/UX design and wireframing.
  • WhatsApp - For team communication and coordination.
  • GitHub - For version control and code collaboration. The repositories on how to deploy the website locally or by using container (Docker Desktop) Access to the version control such as Github. Click HERE.
  • Digital Ocean - I used this to deploy the website to the cloud. I use DG because it is easier to setup the firewall rules and port configuration without too much configuration. Here is the step-by-step process on how I set it up installing the latest version of NPM, Node.js and SQLITE (to handle the backend) Click HERE.
  • Netlify - To show the dummy website without affecting the backend. Just pure front-end to avoid exposing the API keys and without affecting the database. (SQLITE)
  • Google Drive - For file storage and sharing. Access all project-related documents here.

These tools helped us collaborate efficiently, ensuring that tasks were properly tracked, and communication was clear throughout the project lifecycle. The files and resources for this project are available for reference and can be accessed on Google Drive.
